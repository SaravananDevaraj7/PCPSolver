What
----

PCPSolver is a Java application which (tries to) solve instances of
Post's Correspondence Problem, PCP for short
(http://en.wikipedia.org/wiki/Post_correspondence_problem).


Build and run
-------------

1. Clone repository
2. Go to root directory of the project
3. Build executable JAR with "mvn install"
4. Run with "java -jar target/pcpsolver-1.0-SNAPSHOT-jar-with-dependencies.jar"


Dependencies
------------

Java 6
Maven 3
drc.jsearch.1.0-SNAPSHOT.jar
JUnit 3.8.1 for the unit tests
